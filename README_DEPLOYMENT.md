### Шаг 3: Настройка веб-сервера (Nginx)
```nginx
server {
    listen 80;
    server_name sportit.tech www.sportit.tech;
    
    root /var/www/sportit.tech;
    index index.html;
    
    # Редирект с www на без www
    if ($host = 'www.sportit.tech') {
        return 301 http://sportit.tech$request_uri;
    }
    
    location / {
        try_files $uri $uri/ =404;
    }
    
    # Кэширование статических файлов
    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }
}
```

### Шаг 4: Настройка SSL (HTTPS)
```bash

# Получение SSL сертификата

```

## 🚨 ВОЗМОЖНЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ

### Проблема 1: Не отображаются стили
**Решение:** Проверить пути к CSS в index.html
```html
<!-- Должно быть: -->
<link rel="stylesheet" href="style.css">
```

### Проблема 2: Не работают JavaScript
**Решение:** Проверить консоль браузера (F12 → Console)
**Проверить:** Что файл advantages-interactive.js загружен

### Проблема 3: Нет cookies уведомления
**Решение:** Проверить что cookies-popup.css загружен
**Проверить:** Консоль на ошибки загрузки CSS

### Проблема 4: Плохая адаптивность
**Решение:** Проверить viewport в index.html
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```
